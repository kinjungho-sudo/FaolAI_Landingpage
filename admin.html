데이터 저장: 이 브라우저의 localStorage에서 불러왔습니다. Supabase 연동 후 모든 기기의 등록자를 한 곳에서 관리할 수 있습니다.
⚡ Supabase 연동 설정
아래 순서대로 진행하면 3분 안에 설정이 완료됩니다.
Supabase 무료 플랜으로 충분합니다.
STEP 1 Supabase에서 테이블 생성

supabase.com → 프로젝트 생성 → SQL Editor에 아래 코드를 붙여넣고 실행하세요.

create table registrations ( id bigint generated always as identity primary key, email text unique not null, source text, ref text, created_at timestamptz default now() ); alter table registrations enable row level security; create policy "anon_insert" on registrations for insert to anon with check (true); create policy "anon_select" on registrations for select to anon using (true); create policy "anon_delete" on registrations for delete to anon using (true);
STEP 2 API 정보 입력

Settings → API 에서 Project URL과 anon public key를 복사하세요.

✅ 연결 성공! 데이터를 불러옵니다.
